# dl-ui-scaffold

- **Purpose**: Quickly scaffold a React TSX component that matches DiscourseLens styling.
- **Command**: `python .codex/tools/scaffold_ui.py StatCard --out dlcs-ui/src/components`
- **Notes**: Uses `lucide-react` icon, Tailwind-style classes, and outputs to the chosen folder (default `dlcs-ui/src/components`). Use `--force` to overwrite an existing file.
